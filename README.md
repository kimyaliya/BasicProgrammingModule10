## Simple C Calculator

This is a very simple calculator program written in C.
It allows the user to input an operator (+, -, *, /) and two numbers.
The program then performs the selected operation and displays the result.

Features
- Addition
- Subtraction
- Multiplication
- Division (with division-by-zero check)

How to Use
1. Run the program.
2. Enter one operator:
- "+ for addition"
- "- for subtraction"
- "* for multiplication"
- "/ for division"

Enter two numbers when asked.
The program will output the result.
